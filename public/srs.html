Software Requirements Specification (SRS) for Online Registration Form

1. Components

1. 'First Name' text field

1.1 Should accept minimum 4 alphabets and maximum 10 alphabets
1.2 It is a mandatory field
1.3 It should contain a placeholder “First Name”
Error: Minimum allowed characters should be 4, maximum should be 10. Placeholder mistake. Accepts special characters.

2. 'Last Name' text field

2.1 Should accept minimum 4 alphabets and maximum 10 alphabets
2.1.1 It is not a mandatory field
2.1.2 It should contain a placeholder “Last Name”
Error: Not marked as a mandatory field.

3. 'User Name' text field

3.1 It should accept minimum 6 and maximum 20 characters
3.1.1 It is a mandatory field
3.1.2 It should contain a placeholder “User Name”
Error: Displays "UN", maximum accepts 100 characters.

4. E-mail Address text field

4.1 It is a mandatory field
4.2 It should accept valid Email ids only
4.3 It should contain a placeholder “Email”
4.4 It can accept alphabets, numbers, and special characters but space not allowed
Error: Space allowed in the beginning.

5. 'Address' text area field

5.1 It should accept up to 2000 characters
5.2 It is not a mandatory field
5.3 It should contain a placeholder “Address”
5.4 It should accept all special characters
Error: Spelling mistake in placeholder "ADDRDDR'S AFTER PWD".

6. 'Mobile Number' text field

6.1 Should accept 13 digits and minimum 10 digits
Error: Accepts alphabets and special characters.

7. 'Select City' dropdown

7.1 It is not a mandatory field
7.2 It should contain Bengaluru, Mumbai, Chennai, Pune in order
7.3 It should have a placeholder “Select City”

8. 'Password' text field

8.1 It should be alphanumeric and should contain a minimum of one special character
8.2 It is a mandatory field
8.3 It should contain a minimum of 6 and maximum of 10 characters
8.4 It should contain a placeholder “Password”
8.5 It should contain a minimum of one uppercase letter

9. 'Gender' Radio button

9.1 Should select either Male or Female
9.2 It is a mandatory field

10. 'Terms and conditions' link

10.1 Should display terms and conditions page on click
10.2 It is mandatory
Error: GO TO BLANK, should be a checkbox

11. 'Submit' button

Should be present

12. 'Cancel' button

Should be present
